## Visa & Sponsorship 🛂

**Critical for International Applicants to Australia**

### Main Visa Pathways

- [Skilled Independent Visa (189)](https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-finder/find-a-visa?visasearch=skilled%20independent) - PR without sponsor (points-based)
- [Employer Sponsored Visa (482/484)](https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-finder) - Company sponsors you for work
- [Temporary Skill Shortage (482)](https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-finder/find-a-visa) - Short-term skilled worker visa
- [Working Holiday Visa](https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-finder/find-a-visa) - For age-eligible (typically under 35)

### Occupation Lists

- [MLTSSL (Medium Long Term)](https://immi.homeaffairs.gov.au/visas/working-in-australia/skilled-visas/occupations-lists) - Occupations eligible for 189 visa
- [STSOL (Short Term)](https://immi.homeaffairs.gov.au/visas/working-in-australia/skilled-visas/occupations-lists) - Eligible for 482 visa
- Check occupation codes: Software Engineer (261313), Data Analyst (261311), etc.

### Tools & Resources

- [Skill Select](https://immi.homeaffairs.gov.au/skilled-migration-program-report) - Check your points
- [Points Calculator](https://immi.homeaffairs.gov.au/help-support/tools/points-calculator) - Calculate PR eligibility
- [Migration Institute Australia](https://www.mia.org.au) - Find registered migration agents
- [DHA Official](https://immi.homeaffairs.gov.au) - Government source of truth

### Company Sponsorship

Many companies sponsor:
- **Google** - Yes, routinely sponsors
- **Amazon/AWS** - Yes, actively sponsors
- **Microsoft** - Yes, sponsors
- **Meta** - Yes, sponsors
- **Stripe** - Yes, very sponsor-friendly
- **Atlassian** - Yes, sponsors (even for AU/NZ)

Not all roles are sponsorable—ask in application: "Are you able to sponsor visa sponsorship?"

## Job Search Optimization 🔍

**Job Search Strategies**

1. **Start with LinkedIn** - Set job alerts, optimize profile
   - Follow companies you're interested in
   - Post about your job search (gets recruiter attention)
   - Enable "Open to Jobs" status

2. **Direct Company Applications** - Higher success rate for FAANG
   - Apply directly via company careers page
   - Mention visa sponsorship early (if needed)

3. **Recruiter Outreach** - External recruiters can help
   - Search "tech recruiter Australia" on LinkedIn
   - Many specialize in Google, Amazon, Microsoft placements

4. **Networking** - Often faster than applications
   - Coffee chats with people at target companies
   - Tech meetups and events
   - GitHub profile with contributions helps

**Job Search Tools**

- [Huntr](https://www.huntr.co) - Track applications, follow-ups
- [Trello](https://trello.com) - DIY job search organization
- [Notion](https://www.notion.so) - All-in-one workspace
- [JobScan](https://www.jobscan.co) - Match resume to JD
- [Google Alerts](https://www.google.com/alerts) - Monitor company announcements

## Pro Tips for Australian Tech Job Hunters 💡

1. **LinkedIn is Essential** - Australian tech ecosystem is LinkedIn-heavy
2. **Visa Sponsorship Early** - Tell companies upfront if you need sponsorship
3. **ATS Keywords Matter** - Use job description keywords in resume/LinkedIn
4. **Salary Negotiation** - Use Levels.fyi + Glassdoor data for negotiation
5. **Async-First Remote** - GitLab/Figma better for AU timezones than US-heavy companies
6. **Networking Beats Applications** - 40-50% of jobs filled through referrals
7. **Be Patient** - Tech hiring pipelines are 6-12 weeks typically
8. **Update Portfolio** - GitHub + side projects strengthen applications
9. **Skills That Hire** - Python, JavaScript, SQL, AWS always in demand
10. **Contract/Freelance** - Alternative if permanent sponsorship difficult
